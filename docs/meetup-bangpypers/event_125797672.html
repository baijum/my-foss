  • Tell a friend
  • Share

  • Saturday, May 17, 2014

    10:00 AM     to 5:00 PM

  • Intuit Technology Services Private Limited

    Intuit Building 4A 7th floor, RMZ Ecospace Outer ring road Bellandur, Bangalore (map)

    12.924517 77.673775

  • May Meetup: 17/05/2013 10.00 AM to 5.00 PM

    More with python:

    Last few months we had basic python session. This month we will more concept. This will be Intermediate level hands on session.

    Topics:

    • Passing functions as arguments to functions 

    • Decorators 

    • Introduction to classes 

    • Inheritance 

    • Recursion 

    • Unit Testing 

    • Installing third party libraries 

    • Using standard library and third party libraries


    Requirements:

    - Bring your laptop :-)

    - You should know variables, functions, list data structure in python. You can do quick readup from http://kracekumar.com/post/71171551647/introduction-to-python

    - Python 2.7 installed with your favourite text editor.

    - Pip (http://pip.readthedocs.org/en/latest/installing.html, ) and virtualenv installed (http://www.virtualenv.org/en/latest/virtualenv.html#installation)

    Note: If you don't know python, please learn basic before RSVPing :-). We won't spend time teaching basics.

    Instructor:

    - Kracekumar (https://twitter.com/kracetheking)

    Please feel free to write to [masked] or @kracetheking for any help.
