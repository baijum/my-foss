July Bangpypers meetup - Workshop

  • 2 days ago · 10:00 AM
  • ZEOMEGA INFOTECH PVT. LTD.

Writing Idiomatic python by Anand Pillai:

 Python has lot of concepts like list comprehension, dictionary comprehension, iterators to write clean code. These concepts not allow help us to write clean code but are foundations for  data crunching and handling large data.

Topics:

• List/dict comprehension

• Generator comprehension

• Generators

• Iterators

• Itertools (please go through itertools modules docs https://docs.python.org/2/library/itertools.html and http://pymotw.com/2/itertools/)

BangPypers was founded in Feb 2005 by Anand. Huge kudos for him. He is the author of library harvestman. In PyCon India 2013, he gave a talk on Idiomatic Python which was very well received and very well known pythonista.

Requirements:

• Python 2.7.x installed in own laptop.

• Basics of Python like data structures, control flow, functions etc ...
