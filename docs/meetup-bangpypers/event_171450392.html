April Bangpypers Workshop

  • 2 days ago · 10:00 AM
  • Akamai Technologies

April Bangpypers workshop - 19th April, 2014 10.00 AM - 5.00 PM

Audience Level: Beginner level programming experience

Attendees should bring their own laptop and python 2.7 installed.

Agenda:

Learn python by example - Kracekumar (https://twitter.com/kracetheking)

Workshop will focus on learning python through examples. Every concept will be explained by solving small problem. Workshop will cover variables, string, conditional statements, loops, data structures, functions. 

Pre requisite:

1. Laptop with Python 2.7 installed (https://www.python.org/download/releases/2.7/).

2. Text editor setup like Sublime Text (http://www.sublimetext.com/) 

Installation guide:

1. http://docs.python-guide.org/en/latest/starting/install/osx/

2. http://docs.python-guide.org/en/latest/starting/install/win/

3. http://docs.python-guide.org/en/latest/starting/install/linux/


Venue: Akamai Technologies, Outer Ring road, Bellandhur - http://goo.gl/oiXJCw

Contact: Kracekumar - [masked] - [masked].
