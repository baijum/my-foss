jaisen posts:

>> When mounting, it displays: No  Media found..  What is the problem?
>> Can it be resolved?

It can be  resolved.  BTW, are you using the stock  kernel or a kernel
compiled by you ?  The issue may be due to the following reasons,

* Your CD Writer device driver may not have been loaded, test this out by
  doing an `lsmod'.  Maybe you should  paste the output of this to the
  List.

* You may not have Joliet support in the kernel.  Please try to mount
  the CD-R/W  using the command `mount -t vfat <your CD-RW device> /cdrom'
  and ascertain whether you are able to mount it.

* If your  CD writer is  with an IDE  interface, then you  should have
  insmoded the `ide-scsi.o' module.


More info on this has to be articulated by you, only then the List can
help you fix the problem.
