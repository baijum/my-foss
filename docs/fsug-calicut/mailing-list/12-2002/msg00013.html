"Mathrubhumi" (Malayalam Daily) has been spreading false 
information associating Linux with R M Stallman.

There are at least 6 reports I have seen in the newspaper
with the phrase "Richard Stallman's Linux". 

Today, 14th Dec, there is a report in the paper that MS
is considering opening up the source code of their software
fearing pullout by other states from using their software
following MadhyPradesh's decision to use Linux in their
E-Governance. This is absolutely untrue. Microsoft
has been talking about this for more than one year
in what they call "shared source" approach. 

The paper is using Linux as a means of their normal 
"twisting of facts" media trick to create public 
interest. This should be protested against.
