LTSP contains only shell scripts.
It is a nice packaging of things required for booting on NFS mounted file
systems. No special code.
What you do is the following

Run DHCP server , TFTP server, export a directory that will become the
root for remote machines.
The smart thing thay have done is making the exported partition read-only
and
made the write things like /tmp etc. on a ram disk

please see the installation manual at www.ltsp.org


BTW
I have got a LTSP within 15 minutes almost ready. Yes, starting from a
blank hard disk, it takes only that much to
install a GNU/Linux system and get LTSP going with things like
OpenOffice, Koffice and some ttf fonts.
Working on combining GLUE with this distribution on a single CD.
for details see www.peanutlinux.org
