Hi,

Any one heard about Indix? It's a modified X - Window which can handle the Indic scripts.. The modification has been made in original XFree86 4.0.3 source, by National Centre for Software Technology, Mumbai, India.

  As seen in it's release notes,
-----------------------------------------------------------------

- It's a Central Govt. project, funded by Ministry of Information Technology, started in June 2000.

- It has UTF-8 Support, and indic characters are rendered according to Unicode standard.

- It has OpenType font support

- It has Built in Indic Script Shaping Engine. This engine is being used by X server as well as Xlib library. This shaping engine is a shared object (.so file) and can also be used by client programs.
It is capable for script breaking, syllable breaking, reordering, etc. which is needed for complex Indic script processing.

- The IndiX system has built in Indic locale with Unicode (UTF-8) support. This UTF-8 locale can be used by other libraries or applications.
-----------------------------------------------------------------
