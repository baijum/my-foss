I have not seen your earlier posting but the system will stop for long time
while initializing eth0 if you
have an ethercard and did not enter the IP address during installation
If you just press enter there, it assumes DHCP and during boot time
sends out a  DHCP request and waits, until it times out.

If this is the problem,
run 'netconfig'  and  set proper IP address and then restart the system
