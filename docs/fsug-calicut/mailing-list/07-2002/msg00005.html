I think by installing the fonts of the newspapers, 
we can view the websites. (I don't have the permission
to do this in my college)

To install font :

1. Download the TTF font.
2. Place it in a directory.
3. Run 'ttmkfdir' command. (from the same directory)
    $ttmkfdir > fonts.dir
4. Add this font path into the XF86Config-4 (or whatever your
   X configuration file)
5. restart X (Ctrl+Alt+Backspace)

Those who have access to net, please try this and send your
feedback. I have tried some fonts in editors. Now a free
font is available at http://malayalamlinux.sf.net
(GPLed)
