> >
> > Now almost all Malayalam webpages can view,
> > by just copying the fonts to $HOME/.fonts
> > directory (and enter $fc-cache, once)
> Can you explain the detailed steps and other requirements.
> I am running RedHat 7.3 and mozilla 1.3 beta
> How to proceed to read malayalam newspapers

1. pkgconfig http://www.freedesktop.org/software/pkgconfig
2. fontconfig and its related packages
(Xft,Xrender) http://www.fontconfig.org

3. gtk+ 2.x and its related packages (glib,pango,atk) http://www.gtk.org

4. libIDL http://andrewtv.org/libIDL/libIDL-0.6.8.tar.gz

5. Compile the mozilla 1.3 beta source with the following options :

--enable-default-toolkit=gtk2 --disable-gtktest --disable-mailnews --disable-ldap --disable-freetypetest --enab! le-xft --disable-postscript --disable-xprint --enable-crypto --disable-jsd --enable-xinerama --disable-composer --disable-mathml --disable-installer --disable-tests --disable-debug --enable-optimize=-O2 --disable-dtd-debug --disable-pedantic --disable-glibtest

If you are having Redhat 8.0,Mandrake 9.0 etc just compiling mozilla source or installing binary will be enough.
