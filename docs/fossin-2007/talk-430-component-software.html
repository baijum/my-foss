Speaker Name Prashanth N Udupa
Organization VCreate Logic (P) Ltd.
Type Tutorial
Scope Technical
Component Software Design/Development
Abstract Component Software is the order of the day today. Gone are the days where people wrote monilithic software where everything that the software could do would come in one piece. These days it is important to write applications in such a way that it is possible to "breathe life" into deployed applications without having to send a completely new copy of the whole software. Component programming methodologies are important for both open source and commercial software.

KDE already has a fantastic component framework called KParts, which is exhaustive, neat, well tested and widely used. It might be a bit difficult for new KDE developers to get started on KParts. In this tutorial the speaker will introduce a component framework called GCF (Generic Component Framework) - an award winning component framework for Qt applications. GCF is small, light weight and a good starting point to understand component software development methodologies.

For this talk the speaker will come with a sample code that has some functionality written using GCF. While explaining GCF concepts, the speaker will implement new functionality in front of the audience. While the sample code will help save time in showing some basic stuff, the real-time coding helps in understanding how GCF can be leveraged in applications.

One such code demonstration would be integration of the KDE4's Marble Desktop Globe component into VTK Designer 2 using GCF.

Visit http://vcreatelogic.com/oss/gcf to know about GCF .
Pre-requisites Must Know: C++, Qt.
Speaker Profile Prashanth is a developer at heart with experience and expertise in C++, Qt, OpenGL, VTK and OpenHaptics. He is an avid Open Source contributor. a) His Generic Component Framework for Qt Applications won the 3rd place in the world wide Qt developer contest hosted by QtCentre. http://contest.qtcentre.org/results b) He is one of the key developers of the VTK Designer project. He has presented VTK Designer at several platforms like: ICHMI 2004, FOSS.IN/2006, WIPRO and to several other student and professional audiences. VTK Designer 2, due to release in October 2007 was for the most part designed and implemented by him. http://www.vcreatelogic.com/oss/vtkdesigner/vtkdesigner2.html contains some preview videos. c) He has given many talks on Qt. One of the most memorable ones was with Lars Knoll (of KHTML and Trolltech fame) to a group of developers from different companies in Bangalore. Prashanth works for VCreate Logic (P) Ltd, a company that he founded in 2005, which offers design and development consultancy services using C++/Qt, OpenGL, VTK and OpenHaptics as tools.
