An Input Method for Malayalam

By Baiju M — from docs/input-methods.html

Detailed documentation for the Malayalam Inscript keyboard input method, covering:

  • Downloading and installing the XKB file, locale, Compose file, and font
  • Setting up the X Window System font path
  • Compiling the ml_IN locale
  • Configuring compose.dir for chillu character composition
  • Inputting Koottaksharas (conjunct consonants)
  • Inputting old Malayalam characters (U+0D0C, U+0D60, U+0D61)
