Swathanthra Malayalam Computing (SMC)

This Yahoo Group was the original communication channel for the Malayalam GNU/Linux Project, which aimed to translate X, GNOME, and KDE into Malayalam—the language spoken by 38 million people, primarily in Kerala, India.

The group’s description, as captured by the Wayback Machine in 2019, read:

“This project is moved to https://savannah.nongnu.org/projects/smc with a new name Swathanthra Malayalam Computing. Please join the mailing lists there.”

History

2001
Baiju M, a student at NIT Calicut, starts a community called “Malayalam Linux Online”.
December 21, 2001
The malayalamlinux Yahoo Group is created. The goal: translate X, GNOME, and KDE into Malayalam.
January 17, 2002
The Malayalam GNU/Linux project is registered on SourceForge by baijum81, karunakar, malayalamlinux, and shaji.
2002–2003
Peak activity period. Discussions cover Malayalam in GTK2, Unicode encoding for Malayalam, Chillaksharam properties, Varamozhi input method, technical term translations, and collaboration with Indic computing standards groups.
October 2002
The community adopts the name Swathanthra Malayalam Computing (SMC) and moves to Savannah. The Yahoo Group remains but activity shifts to SMC mailing lists.
2010
SMC formalizes as a registered non-profit charitable society.
April 2017
Archive Team captures the malayalamlinux Yahoo Group via the Yahoo Groups API. The data is stored as a 617 KB WARC file.
December 2020
Yahoo Groups shuts down permanently. All content deleted.
April 2026
This partial archive is recovered from the Wayback Machine. 53 message subjects extracted from cached listing pages.
