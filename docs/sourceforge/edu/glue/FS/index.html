GLUE is a collection of documentation and software packages to help you to get started into the world of Free Software. It contains articles explaining the philosophy of Free Software and the advantages of it. The most popular Free Software platform is the GNU/Linux operating system that is available under the GNU General Public Licence. GNU/Linux distribution CDs are available along with several computer magazines and books. There are thousands of other programs available under GPL and similar licences. We have included some of them on this CD along with installation instructions and user manuals.

How to use GLUE CD

  1. You can browse this CD under any operating system to read the documentation and run the java applets.
  2. You can install the packages from the CD to a computer. You need to install Linux first. Most of the RPM files are tested on a RedHat 7.3 system. The 'setupglue' ( menu driven version is setupglue.menu) program can be used for this. Skip the initial sections on Terminal Server while running it.
  3. You can use this CD to convert your computer into a Terminal Server using this CD. Run the 'setupglue' program from the CD for this. For more details see the Sections on SchoolNET and LTSP.
  4. If you like to utilize this fully, create a directory named 'glue' inside the Web Server Document Root directory, 'var/www/html' for RedHat, copy the contents of this CD to it and run 'setupglue' from the same directory. This will make the computer a Terminal Server, Install a light and fast Window Manager and make all the installed packages available under the StartMenu( if you choose to login under icewm). See SchoolNET and LTSP for details

Note: The script will install most of the packages but is not very smart. You can manually install the RPM packages from the RPMS directory and the tar files from the TGZ directory. Bbrowse the CD for installation instructions for each package.

Purpose of this work

Information Technology and Computers have become an integral part of the Indian society also. We have already seen how it can improve our lives. A good example is our Railway reservation and information system and there can be hundreds of other areas like that. Progress of a country depends a lot on how it accepts and assimilates technology. Acquiring technology without mastering it may bring the benefits of it quickly but will eventually result in technological slavery and wastage of national resources. For stable long term benefits, computerization in the country should be done based on the knowledge we acquire and systems we develop. Not by readymade solutions provided by Multinational Companies driven purely by commercial interests and having the clout to control even the most powerful governments. For that to happen, we need to learn the basics properly. Free Software is a powerful tool for that and this collection is to make your road to Free Software easier.

Free Software

In the early days of computers the software used to come with the hardware and the source code was provided so that the user could make changes to it. Later the companies started selling software on very restricted license terms. After paying the customer only gets a permission to run it on a single computer and the source code is not given. Many programmers who are accustomed to the concept of sharing found it difficult to accept the idea of keeping things secret. The most important figure among them may be Richard Stallman of MIT who started decveloping a free unix-like operating system in 1984, under a project named GNU. GNU is a recursive acronym that stands for Gnu's Not Unix. He started writing the components of an OS and started distributing it. In 1985 he started Free Software Foundation, which has spread to many countries now. The Indian wing of FSF started in Jul-2001 and operating from Trivandrum. To know more about Stallman read Free as in Freedom. The GNU project produced most of the components except the Kernel. Linus Torvalds, a computer science student from Finland, started working on a kernel in 1991 and was joined by a group of people on the Internet. As a result the new system GNU/Linux was born.
